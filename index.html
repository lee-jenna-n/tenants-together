Email Script - Tenants Together Resource Directory Outreach

How to use this page:

  1. Check the Local Resource Directory volunteer sheet to figure out which organizations to email.
  2. Add the volunteer name (i.e. your name) and the organization's name in the text inputs on the left. These names will be added to the generated emails on the right.
  3. Click the "Copy to clipboard" button to copy the correct email (i.e. first or second), then paste it into Gmail and send it. You can also use a normal copy+paste if you prefer.
  4. Update the Local Resource Directory volunteer sheet to show that you have contacted the organization.
